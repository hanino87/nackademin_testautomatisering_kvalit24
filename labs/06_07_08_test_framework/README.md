# Lab 06,07,08 - Structure a Test Framework


## Prepare a virtual environment

mac
```shell
rm -rf .venv
python3 -m venv .venv ; pip install --upgrade pip ; source .venv/bin/activate


pip install -r requirements.txt


playwright install
```

windows - powershell
```shell
rmdir .\.venv\
python -m venv .venv
. .\.venv\Scripts\activate

pip install -r requirements.txt


playwright install
```
